# Transfer Protocol

## Transfer Trigger Keywords
- "TRANSFER_TO_RANDY" (literal text in transcript)
- Caller explicitly asks to speak with Randy
- "Can I talk to Randy?"
- "Is Randy available?"
- "I need to speak with Randy"
- "Puedo hablar con Randy?"
- "Necesito hablar con Randy"

## Transfer Procedure

### OPS Assistant
When transfer is triggered:
1. Say: "Let me transfer you to Randy."
2. Stop speaking immediately.
3. Server handles the actual transfer.

### TECH Assistant
When transfer is triggered:
1. Say: "Transferring you to Randy now."
2. Stop speaking immediately.
3. Server handles the actual transfer.

## Transfer Conditions
- Complex technical issues beyond troubleshooting scope
- Customer escalation requests
- Business/pricing negotiations
- Complaints or disputes
- Anything requiring management decision

## DO NOT Transfer For
- Standard intake calls
- Routine job logging
- Basic troubleshooting
- Status inquiries
- Scheduling questions

## Post-Transfer
- Server handles the WebSocket message
- No assistant action required after announcing transfer
- Call state is preserved for Randy's reference

# Download Plan

## What to Download from Each OEM

This document specifies exactly what should be downloaded from each OEM portal to build a complete ADAS + OEM repair knowledge base.

---

## Download Priority Levels

**Priority 1 (Critical):** ADAS calibration procedures, position statements
**Priority 2 (High):** Service manuals for ADAS-equipped models, TSBs
**Priority 3 (Medium):** Wiring diagrams, diagnostic procedures
**Priority 4 (Low):** General service information, owner manuals

## Sources Requiring Manual Download vs. API/RSS

**Manual Download Required:**
- All OEM portals (no API access)
- Third-party platforms
- NASTF documentation

**RSS/Email Alerts Available:**
- TSB notifications (some OEMs)
- I-CAR RTS updates
- ALLDATA/Mitchell1 update notifications

**No Automated Collection:**
- OEM portals generally prohibit scraping
- Manual download during subscription periods only
- Document source and date for all archived materials

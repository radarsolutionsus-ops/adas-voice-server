# OEM Folder

## Purpose
Contains brand-specific folders for each OEM with service information, ADAS procedures, and programming details.

## Folder Structure

Each brand folder contains:
- **Portals/** - Portal URLs, login info, subscription details
- **ADAS/** - Calibration procedures specific to this brand
- **Programming/** - Diagnostic software and VCI requirements
- **PDFs/** - Downloaded documentation (during active subscription)
- **Notes/** - Brand-specific quirks and known issues
